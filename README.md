# Dokumentasi Rangkaian 7 Logic Gate

Repositori ini berisi penjelasan, dokumentasi simulasi, dan link Tinkercad untuk 7 gerbang logika dasar (AND, OR, NOT, NAND, NOR, XOR, XNOR).

---

## 1. Gerbang AND
* **Penjelasan:** Gerbang AND akan menghasilkan output 1 (lampu menyala) hanya jika semua inputnya bernilai 1. Jika ada satu saja input 0 atau salah satu tombol OFF, maka outputnya 0 (lampu mati).

* **Penjelasan:** Gerbang OR akan menghasilkan output 1 (lampu menyala) jika ada satu input yang bernilai 1 atau salah satu tombol ON.

## 3. Gerbang NOT
* **Penjelasan:** Gerbang NOT akan menghasilkan kebalikan dari input yag diberikan, Jika input 1 atau ON diberikan. Maka outputnya 0 (lampu mati), begitupun sebaliknya.
